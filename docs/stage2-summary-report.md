## 4. 收敛方法（本轮建立）

富 Ta/多态角落靠以下机制达到 100%：
- **逐点体积感知 EFGS**（Σconc·系数 + sws）+ 外推余量
- **离群能量守卫**：聚簇后保留"点数≥3 的最低能簇"（=基态），挡住亚稳态/伪点
- **收敛覆盖**：`Ta≥30 → depth 0.80 + AMIX 0.02`，外加显式列表（9 个低 Ta 但陷入亚稳/散点的合金）
- **B0 上限 350 GPa** 挡非物理拟合

关键物理认识：难熔 HEA 中**电子态双稳普遍存在**；EOS 必须拟合**基态簇**。**AMIX（电荷混合），而非 EFMIX，决定 SCF 落入哪个电子态**——AMIX=0.02 把多态压成单一基态。

## 5. 已知局限（使用数据时注意）

1. **R² 对精细窗口意义有限**：±1.5% 窗口能量仅变 1–2 mRy（∝B0），软合金 R² 天然偏低——**用 B0 物理性 + sws0 是否被夹住判断，别单看 R²**。
2. **B' 和 Gruneisen 不可用**：Morse 把 B' 钉在 4；BM 在窄窗口下 B' 约束不住（同一 EOS，B'=4 与 B'=−0.26 曲线重合）。要真 B' 需在粗窗口（stage1 ±3%）做 BM。
3. **B0 绝对值**：W/Mo 富集算得准（W91Re9=306 vs 实验 310）；**富 Nb 中位 164 GPa vs 实验 ~170，基本合理**（个别如 Nb96 偏软至 146，属 EMTO 球近似 + PBE 对 Nb 的轻微低估）。
4. **9 个略偏心**（\|dSWS\|0.03–0.11，多为软合金）：sws0/B0 均可用，浅阱几何收敛慢所致，非坏拟合。
